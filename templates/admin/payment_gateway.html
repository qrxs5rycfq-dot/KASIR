Cara Setup Midtrans

1

Buat akun di midtrans.com dan verifikasi merchant.

2

Buka Dashboard → Settings → Access Keys untuk mendapatkan Server Key & Client Key.

3

Masukkan keys ke file .env dan restart aplikasi.

Cara Setup BRI QRIS

1

Daftar di developers.bri.co.id sebagai merchant partner.

2

Dapatkan Client ID, Client Secret, Merchant ID, dan Terminal ID dari BRI.

3

Generate RSA key pair, upload public key ke BRI developer console.

4

Masukkan credentials ke .env dan restart.
