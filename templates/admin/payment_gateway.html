Cara Konfigurasi

Penting!

Konfigurasi payment gateway dilakukan melalui file .env

1. Buat akun Midtrans

Kunjungi midtrans.com dan buat akun merchant.

2. Dapatkan API Keys

Login ke dashboard Midtrans, buka Settings > Access Keys untuk mendapatkan Server Key dan Client Key.

3. Edit file .env

# Midtrans Configuration
MIDTRANS_SERVER_KEY=your_server_key_here
MIDTRANS_CLIENT_KEY=your_client_key_here
MIDTRANS_IS_PRODUCTION=false

4. Restart aplikasi

Setelah menyimpan file .env, restart aplikasi untuk menerapkan perubahan.

Perhatian

  • Gunakan mode Sandbox untuk testing
  • Ganti ke Production setelah siap live
  • Jangan bagikan Server Key ke siapapun
