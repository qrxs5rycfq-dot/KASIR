Panduan Integrasi

Penting!

GrabFood, GoFood, dan ShopeeFood menggunakan sistem Merchant Integration yang memerlukan pendaftaran sebagai mitra resmi. Setelah terdaftar, Anda akan mendapatkan akses untuk mengkonfigurasi webhook yang akan mengirim pesanan otomatis ke sistem kasir ini.

GrabFood

  1. 1. Daftar sebagai merchant di GrabMerchant Portal
  2. 2. Akses menu Developer Settings
  3. 3. Masukkan Webhook URL di atas ke kolom Order Notification URL
  4. 4. Salin Webhook Secret dari portal
  5. 5. Set environment variable:
    GRABFOOD_WEBHOOK_SECRET=secret_anda
  6. 6. Pesanan akan otomatis masuk dan tercetak!

GoFood

  1. 1. Daftar sebagai mitra di GoBiz Dashboard
  2. 2. Akses API Integration settings
  3. 3. Masukkan Webhook URL di atas ke kolom notification URL
  4. 4. Salin API Key/Secret dari dashboard
  5. 5. Set environment variable:
    GOFOOD_WEBHOOK_SECRET=secret_anda
  6. 6. Pesanan akan otomatis masuk dan tercetak!

ShopeeFood

  1. 1. Daftar sebagai merchant di Shopee Partner Portal
  2. 2. Akses Webhook Configuration
  3. 3. Masukkan Webhook URL di atas ke kolom Order Webhook
  4. 4. Salin Webhook Secret dari portal
  5. 5. Set environment variable:
    SHOPEEFOOD_WEBHOOK_SECRET=secret_anda
  6. 6. Pesanan akan otomatis masuk dan tercetak!

Environment Variables

Tambahkan variable berikut di file .env atau environment server Anda:

# GrabFood Integration
GRABFOOD_WEBHOOK_SECRET=your_grabfood_secret_here
GRABFOOD_STORE_ID=your_store_id

# GoFood Integration
GOFOOD_WEBHOOK_SECRET=your_gofood_secret_here
GOFOOD_STORE_ID=your_store_id

# ShopeeFood Integration
SHOPEEFOOD_WEBHOOK_SECRET=your_shopeefood_secret_here
SHOPEEFOOD_STORE_ID=your_store_id

Auto-Print

Setiap pesanan dari platform food delivery akan otomatis dicetak melalui Printer Station. Pastikan Printer Station tetap terbuka di browser untuk menerima pesanan secara otomatis. Setiap pesanan dicetak 2 rangkap (kasir + dapur).

Multi-Cabang

Untuk multi-cabang, setiap cabang harus menggunakan Webhook URL terpisah yang sama, tetapi dengan header X-Branch-Code yang berisi kode cabang (contoh: PUSAT, CBG-S). Jika tidak ada header, pesanan akan masuk ke cabang Pusat.
